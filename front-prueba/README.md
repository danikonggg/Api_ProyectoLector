# API Lector – Frontend React

App React para probar la API Lector. Vista por rol: Admin, Director, Maestro.

## Inicio rápido

```bash
cd front-prueba
npm install
npm run dev
```

Abre http://localhost:5173

## Configuración

- **URL API**: En el login indica la URL (por defecto `http://localhost:3000`).
- **CORS**: El backend debe permitir `http://localhost:5173` si la API está en otro puerto.

## Build

```bash
npm run build
npm run preview
```
